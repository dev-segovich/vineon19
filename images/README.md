# Imágenes del Proyecto Vine Phase II

Coloca las dos imágenes del proyecto en esta carpeta con los siguientes nombres:

- **image1.jpg** - Fachada Principal / Primera vista
- **image2.jpg** - Vista Exterior / Segunda vista

Las imágenes se mostrarán en un carrusel automático en la sección hero de la página.

## Especificaciones recomendadas:
- Formato: JPG o WebP
- Resolución: Mínimo 1000x1200px (para buena calidad en pantallas grandes)
- Relación de aspecto: 4:5 (preferible)
- Tamaño de archivo: Optimizado para web (200-400KB máximo)

El carrusel cambia automáticamente cada 5 segundos, y los usuarios pueden navegar usando:
- Clic en los indicadores de puntos
- Flechas del teclado (← →)
